# Hospital API

## Description

This is an API for a hospital system for testing and quarantine of COVID-19 patients. The API allows doctors to log in, register patients, and create patient reports.

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
